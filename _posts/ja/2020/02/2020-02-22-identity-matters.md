はじめまして、のらです。ポエムっぽい文章を書くのは久しぶりなので少し緊張しています。

## note をはじめたきっかけ

調べ物をしていたら note の記事がヒットして、なんとなく自分の ID を確保しておきたかったという些細なきっかけです。

もともと note というサービスがあることは知っていて、その当時からデザインがとてもおしゃれだなあと感心していました。

とはいえ自分は、文章を書くプラットフォームは自分で創り出すのが至高であるという少し変わったこだわりを持っています。そのため、ブログは自分が管理しているサーバで、独自ドメインで公開しています。

note は厳密にはブログではないかもしれませんが、文章を書くというコンテキストにおいては同一のものと個人的には認識しており、すでにブログを自前で用意している自分にとっては、今さら note に移る必要性は感じていません。

しかし、この美しいデザインを改めて見ると、やはり感銘を受けざるを得ないわけです。おしゃれなものやきれいなものを見ると、つい手にとってみたくなる性分なので、こうした素晴らしいプラットフォームには、使う気がなくてもとりあえずアカウントだけは作っておきたいという気持ちになったわけです。

それと、自分は極度にアイデンティティを重んじる性格なので、Web サービス等で自分の ID が取れるかどうかをものすごく気にします。自分の ID が取れないだけでそのサービスを使う気が失せてしまうくらい、病的に ID を重要視しています。

久しぶりに note の記事を見たときに、なんとなく自分の ID が取れるかどうか確認したら取れそうだったので、誰かに取られてしまう前にとりあえず取っておこうと思いアカウントを作った次第です。こんな感じで、特に使う予定もないのに、おしゃれなデザインをしているからとか、自分の ID がすでに取られていたら嫌だから、という理由だけでアカウントを作ることがよくあります。

すでに自分用のブログを持っているので、今後 note で何か記事を書くということはないかもしれませんが、このおしゃれな場で自分の思いを発信したいという気持ちになったらまた記事を公開するかもしれません。とりあえず現時点では、ID 確保のためにアカウントを作って、せっかくだから記念に 1 本だけ記事を書いてみようと思った次第です。

## よもやま話: ドメインへのこだわり

そういえば、自分が note を知ったころは、ドメインが [note.mu](https://note.mu) だった気がするのですがいつの間にか [note.com](https://note.com) になっていたんですね。知らなかった。

自分も過去に [noraworld.com](https://noraworld.com) というドメインを 30 万円ほど支払って買い取った過去があるので、ドメイン名に対するこだわりはとてもよくわかります。これも ID に対するこだわりと同じですね。

## おわりに

先ほど、自分はブログを持っているので note に記事を書くことはないかも、と言いましたが、実はもう 1 年以上ブログの更新をサボっているんですよね......。ブログとは別に日記も書いていたんですが、それも去年の末あたりにやめてしまって、それっきりになっています。でも、久しぶりにこうして文章を書いていると、やっぱり文章を書くのって楽しいなって気持ちになりました。それを再認識させてくれただけでも note には感謝したいと思います。

これが最初で最後の記事になるかもしれませんが、気が変わってまたここに記事を書くことがあったときは改めてよろしくお願いします。
