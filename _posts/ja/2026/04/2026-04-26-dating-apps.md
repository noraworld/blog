### 自己紹介文
<!-- ↓↓↓↓↓ コピペはここから ↓↓↓↓↓ -->
はじめまして！

休日は、瞑想・ワークアウト・読書・英語学習などを日課にしつつ、note に自分の考えを書いたり、趣味でプログラミングしてツールを作ったりしています。

かなりインドア寄りで、家でYouTubeを見たり静かに過ごしていることが多いです。  
一方で、自分にない世界を見せてくれる人にも惹かれるので、カフェやお出かけに連れ出してくれるような方とも仲良くなれたら嬉しいです ☕

性格はわりと落ち着いているほうだと思います。  
人見知り気味で、自分からグイグイ話しかけるタイプではないですが、好きな話題になるとつい熱く語ってしまうことがあります。笑

一般的な「普通」に縛られるよりも、お互いを先入観なく理解し合える関係に憧れています。  
自然体でいられて、「この人といると安心するな」と思えるような関係が理想です。

テキストのやり取りを延々とするより、実際に会ってゆっくり話すほうが好きです。  
お互いの時間を大切にしつつ、一緒にいる時間はちゃんと楽しめたらいいなと思っています！

よろしくお願いします 😊

### 備考
* Omiai、Pairs、with、タップルは最大 1,000 文字まで、Tinder は最大 500 文字まで
* 段落を改めずに改行する場合は行末に半角スペースを 2 つ入れる
    * `<br>` タグでも改行できるがコピペする際に削除しなきゃいけなくてめんどくさいので不採用
        * 行末の半角スペースは目立たないのでマッチングアプリ内ではそのまま使って問題なし
    * VS Code だと自動で行末のスペースを削除するように設定されているので編集する際は注意すること
        * 基本的にマッチングアプリは iPhone で利用してプロフィール編集時に GitHub アプリで commit / push するため問題ないはず
        * VS Code で編集する場合は消えた行末スペースの差分に対して `git restore -p` すれば良い
            * これのためだけに VS Code に例外の設定を入れるのも微妙なので、ちょっと手間だけど仕方ない
* ChatGPT 等を利用した場合の共有リンクや参考にした保存しておきたいサイトなどは YAML ヘッダの `references` にまとめる
* コミットメッセージは日本語で書く
    * 一新した場合やちょっと更新した場合は「マッチングアプリのプロフィール（テキスト）を一新・更新」などで良い
